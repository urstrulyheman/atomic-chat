# Orca Chat Coin UI

React/Vite frontend wired to the FastAPI MVP backend.

## Run

```bash
cd orca-chat-ui
npm install
npm run dev -- --port 5173
```

Open:

```text
http://127.0.0.1:5173
```

The backend should be running at:

```text
http://127.0.0.1:8000
```

## Demo

1. Log in with any phone number.
2. OTP is auto-filled from the dev OTP response.
3. Click refresh/seed if no contacts appear.
4. Start a chat from Discover.
5. Send a paid message.
6. Watch spendable ORCA, locked rewards, gas, transactions, and admin metrics update.
